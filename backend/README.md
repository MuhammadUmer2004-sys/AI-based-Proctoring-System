# Backend (Flask API)

## Setup Instructions
1. Create a virtual environment:
```
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
```
2. Install dependencies:
```
pip install -r requirements.txt
```
3. Run server:
```
python app.py
```
4. Make POST requests to `http://localhost:5000/video_feed`
